# Stride Status

Use this skill when the user wants a current snapshot of the work.

## Use when

- The user asks what is currently specified.
- The user wants handoff context.
- The user wants a quick project state check.

## Do

- Summarize the current spec.
- Point to the latest handoff.
- Highlight the durable project facts.
